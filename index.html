Description

In this workshop, you’ll learn enough HTML/CSS to create (and understand how to edit) your very own microsite. We’ll show you how to get your new website hosted for free on Github Pages, how to attach a custom domain name to your newly launched site and how to create your own online community (including how to embed a signup form) using platforms like Slack, Facebook, or Mailchimp. Plus, the technical skills you’ll learn can be applied to other parts of your work, allowing you to master tasks like creating email templates and modifying WordPress sites.

This training is an introduction to web development for online organizers who want to learn how to launch their own microsite for free. No prior experience with HTML, CSS or Git is needed.

Setup Instructions (Do ASAP!)

  1. Join the Facebook group at https://www.facebook.com/groups/lindatraining .
  2. Create a github account, and download Git from https://git-scm.com/download/win if you're on a Windows machine and https://git-scm.com/download/mac if you're on a mac machine.
  3. Download Sublime (https://www.sublimetext.com/) if you don't already have a favorite text editor.

Setup Instructions, PART II (We'll go over this at the beginning of the workshop)

  1. Fork this repository by going to https://github.com/lpatmo/make-your-own-microsite-without-spending-a-dime and clicking on the "fork" button. Once it's complete, you'll see a copy of this repository on your own account.
  2. Open up your Terminal or PowerShell. (On a mac, find the Terminal in Applications -> Utilities)
  3. Navigate to your file path of choice using pwd and then ls and cd on Mac, or dir and cd on Windows. The former command lists all of the files in your current file path; the latter command lets you "change directory" into the folder of your choice.
  4. `git clone` your repository into a folder of your choice by typing the following command in your terminal: $ git clone https://github.com/lpatmo/make-your-own-microsite-without-spending-a-dime.git microsite. (Replace "microsite" with another name of your choice if you'd like to name your folder differently. Try to keep this name short!)
  5. Click to open up the project *folder* in your text editor of choice.
  6. Right click and grab the URL of the index.html page of your project. Paste it into a blank web browser tab.
